# TUI Setup (Textual)

## Install
From your repo root (recommended in a venv):
```bash
pip install textual
```

## Run
```bash
python -m creative_os.shell accounting
```

## Keys
- Space: run recommended next action
- m: toggle SAFE/GUIDED/ALL
- p: plan preview (for selected action)
- Enter: apply selected action
