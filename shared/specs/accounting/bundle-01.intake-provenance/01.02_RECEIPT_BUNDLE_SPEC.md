# Receipt Bundle Specification (v1.1)

## Definition
A ReceiptBundle is the atomic unit of accounting evidence.

## Directory Layout
```
bundles/<bundle_id>/
  originals/<sha256>_<filename>
  source_messages/<message_id>.eml
  extracted/extracted_metadata.json
  provenance/provenance.json
```
## Identity
`bundle_id = sha256(concat(attachment_sha256s))`

## Required Metadata
- bundle_id
- created_at
- sources[]
- attachments[] (hash, filename, size)
- currency
- total_amount

## Supersession
Corrections create a new bundle that references the prior bundle_id.
